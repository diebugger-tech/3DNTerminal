# 3DNTerminal v0.3.7

> *Less app, more atmosphere. More inclusive, more alive. Secure by design.*

`#rust` `#cosmic` `#terminal` `#barrierefrei` `#security` `#ai-ready`

A floating, translucent terminal built with Rust and libcosmic, designed for the COSMIC Desktop.

---

## 🛡️ Secure AI-Ready Architecture

3DNTerminal is built for the future of AI-assisted coding.
- **AI-Firewall**: Built-in protection against malicious command execution.
- **Human-in-the-Loop**: Every automated action requires manual approval. No "shadow commands" in your PTY.
- **Privacy First**: Designed for local AI integration (Ollama).

## 🎨 Immersive Deep Theming
- **Glass Ghost Mode**: Ultra-transparent UI for maximum desktop immersion.
- **Unified Icons**: High-fidelity Unicode icon system (Classic/Apple/Transparent) for 100% visual consistency.
- **Adaptive Geometries**: UI buttons morph their shapes (Brackets, Octagons, Circles) based on your active theme.
- **Fluid Skill Sliders**: Redesigned, physics-aware sliders for granular environment control.

## 🌈 Accessibility First (Barrierefrei)

- **Tremor Compensation**: Integrated low-pass filter to smooth cursor interaction.
- **Vision Filters**: Real-time filters for Protanopia, Deuteranopia, and Tritanopia.
- **Smart UI Management**: Auto-closes overlays and menus in compact window states to avoid visual clutter.
